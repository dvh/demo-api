# Nederlandse bieren brouwerij API

Deze API bevat alle brouwerijen in Nederland met bijbehorende coordinaten Rijksdriehoek

Ga naar [https://bier-api.vnb.nl](https://bier-api.vnb.nl) voor meer informatie.

## Features
Deze codebase bestaat uit de volgende features:

- Brouwerijen Endpoint
- Bieren Endpoint
